# Создание игры на pygame
## Жанр игры
Платформер
## Составляющая игры
При запуске игры появляется главный экран на котором находится кнопка "старт" и рекорды прошлых игроков. Чтобы выиграть нужно пройти два уровня, смерть - это поражение. На финальном окне выводится итог игры и результат времени.
## Используемые библиотеки
pygame
